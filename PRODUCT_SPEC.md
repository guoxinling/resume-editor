# 简历编辑器 — 产品规格书

## 概述

一个 Markdown 驱动的双语简历编辑器，提供实时 A4 预览和 PDF/PNG 导出。

## 功能清单

### 📥 输入
- **Markdown 导入**：解析结构化 Markdown 简历文件，自动填充所有模块
- **手动编辑**：表单化编辑个人信息、概述、工作经历、AI项目、教育、技能、语言
- **草稿系统**：IndexedDB 持久化，支持保存/加载/删除多个草稿
- **自动保存**：编辑后 2 秒自动保存至 IndexedDB，启动时自动恢复

### 🌐 双语
- **中文/英文切换**：一键切换编辑界面和预览的显示语言
- **双字段设计**：关键字段均支持中文+英文独立输入（标题、描述、日期等）

### 📋 数据模块
| 模块 | 字段 |
|------|------|
| 个人信息 | 姓名/英文名、电话、邮箱、所在地、年龄、个人网站、LinkedIn |
| 个人概述 | 中文+英文概述段落 |
| 工作经历 | 公司、职位、时间、描述条目（支持多条 bullet） |
| AI 产品实践 | 表格形式：产品、方向、描述、技术栈 |
| 教育背景 | 学校、学位、专业、时间、亮点条目 |
| 专业技能 | 分类+技能列表 |
| 语言能力 | 多语言列表 |

### 👁️ 预览
- **A4 实时预览**：右侧面板实时 A4 比例渲染
- **双语切换**：预览随语言切换实时更新内容

### 📤 导出
- **PDF 导出**：@react-pdf/renderer 生成，Noto Sans SC 字体，A4 版面
- **PNG 导出**：html2canvas 2x 高清截图
- **导出前验证**：姓名+电话/邮箱必填校验

### 🔐 账号与商业化
- **Supabase Auth**：邮箱注册/登录，JWT token 鉴权
- **点数系统**：新用户注册送 20 点，按功能差异化扣费
- **支付**：ZPay 聚合支付（支付宝/微信），3 档套餐（¥9.9/19.9/39.9）
- **后端鉴权**：Service Role Key 验证 token → 扣点 → AI 调用 → 失败退款
- **点数扣费标准**：

| Feature | 扣点 | 说明 |
|---------|------|------|
| wizard | 1 点 | AI 对话式创建简历 |
| polish | 1 点 | AI 经历润色 |
| adapt | 3 点 | 岗位适配分析 |
| interview | 3 点 | 10 题面试预测 |
| translate | 3 点 | 中英全文翻译 |
| import_parse | 5 点 | PDF/图片导入解析 |

### 🏗️ 技术栈
- React 18 + TypeScript + Vite
- Zustand 状态管理
- Tailwind CSS v3
- Dexie.js (IndexedDB)
- @react-pdf/renderer (PDF)
- html2canvas (PNG)
- Supabase (Auth + PostgreSQL)
- ZPay 聚合支付

### 🎨 设计系统
- 完整 Design Token 定义（colors/typography/spacing）
- Tailwind 自定义值接入
- 4px 网格间距系统

### ✅ 字段验证
- 姓名：必填
- 电话/邮箱：至少填写一个
- 内联错误提示 + 导出前拦截
